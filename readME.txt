Приложение выдачи уникального инкрементного идентификатора в разрезе
нескольких групп

Фреймворк, который используется для обмена сообщениями между клиентом и сервером - gRPC
Систем управления базами данных - SQLite 
Работа с базами данных через Entity Framework Core

Генерируемый индекс имеет префикс из заглавных букв английского алфавита и цифровой код (Например A4, BB12, AB434 и т.п).
По полученному индексу можно запросить факт исполнения.

При запуске сервера и далее каждые 90 000 милисекунд (полторы минуты) идентификаторы сбрасываются. Этот параметр настраивается в appsettings.json (ResetInterval).

Желательно настроить в Visual Studio совместный запуск сервера и клиента нажав ПКМ на решение и выбрав "Назначить запускаемые проекты..."; далее - "Несколько запускаемых проектов" и выбрать запуск сервера, затем клиента.
